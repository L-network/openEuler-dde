# autotest

#### 介绍

DDE及预装应用自动化测试仓库(包括测试框架及测试用例)。

#### 目录结构

```
├── docs         #  文档目录
├── libs         # 库文件
├── README.md    # 自动化整体说明文档
├── run.py       # 主入口脚本
├── source       # resource文件
│   ├── app_resource
│   └── dde_resource
└── testcases    # 自动化用例
    ├── app
    └── dde
```

#### 主要应用的库介绍（模块）

* robotframework  整个框架基于robotframework基础框架

* pyautogui  模拟键盘、鼠标对图形化界面进行操作

* tesseract-ocr  自动化过程用到的图像上文字识别出并提取

* scrot 截图工具

* dogtail 使用可访问性（A11Y，Accessibility）技术与桌面应用程序通信的图形用户界面GUI测试工具和自动化框架
