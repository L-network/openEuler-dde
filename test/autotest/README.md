# autotest

#### 介绍

DDE及预装应用自动化测试仓库(包括测试框架及测试用例)

#### 目录结构

```
├── docs         #  文档目录
├── libs         # 库文件
├── README.md    # 自动化整体说明文档
├── run.py       # 主入口脚本
├── source       # resource文件
│   ├── app_resource
│   └── dde_resource
└── testcases    # 自动化用例
    ├── app
    └── dde
```
