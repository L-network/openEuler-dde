# 标题：文本编辑器_帮助信息
* 优先级：2
# 前置条件：
1.系统中存在管理员帐户TEST，且TEST帐户已登入DDE桌面，成功启动文本编辑器
2.命令行查询文本编辑器版本：rpm -qa |grep deepin-editor

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------- | -------------------- |
| 1    | 点击文本编辑器右上角菜单中的“帮助” | 成功弹出文本编辑器“帮助”窗口 |
| 2    | 检视帮助窗口的信息 |  显示文本编辑器的帮助信息 |
| 3    | 任意点击其中的标题 |  可跳转到相应的具体帮助信息界面 |
