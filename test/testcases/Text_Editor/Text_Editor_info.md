# 标题：文本编辑器_关于信息
* 优先级：2
# 前置条件：
1.系统中存在管理员帐户TEST，且TEST帐户已登入DDE桌面，成功启动文本编辑器
2.命令行查询文本编辑器版本：rpm -qa |grep deepin-editor

| 编号 | 步骤                                                 | 预期                 |
| ---- | :----------------------------------------------------| -------------------- |
| 1    | 点击右上角设置按钮，查看列表显示                     | 下拉列表显示：帮助、关于|
| 2    | 点击帮助                                             | 打开文本编辑器的帮助页面|
| 3    | 打开文本编辑器，直接按F1键                           | 打开文本编辑器的帮助页面 |
| 4    | 点击关于 | 显示软件详细信息，"统信UOS“logo显示正确，网址www.chinauos.com显示正确|
