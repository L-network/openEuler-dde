# 打开剪贴板

前置条件
剪切板中有文本，图片，文件信息

| 编号 | 步骤                                                   | 预期                     |
| ---- | :----------------------------------------------------| ------------------------ |
| 1    | 输入“Alt+Ctrl+V” | 唤出剪贴板                           |
| 2    | 唤出剪贴板        | 依次展示“剪贴板，全部清除，剪贴板内容(文本，图片，文件)”   |
