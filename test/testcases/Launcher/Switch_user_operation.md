# 切换用户操作

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------| -------------------- |
| 1    | dock栏点击电源图标 | 打开关机界面 |
| 2    | 点击切换用户 | 当前已登录用户会有√显示 |
| 3    | 在切换用户界面点击空白 | 回到之前用户的锁屏界面 |
| 4    | 点击切换用户 | 进入切换用户页面 |
| 5    | 点击ESC键 | 回到之前用户的锁屏界面 |
