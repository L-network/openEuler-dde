# 启动器缩放显示

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 点击启动器图标 | 进入启动器界面 |
| 2    | 点击右上角缩放按钮 | 启动器可正常进入全屏模式或恢复默认 |
| 3    | 向上或向下滑动鼠标滚轮 | 启动器全屏状态下可实现翻页 |
| 4    | 全屏状态下拖动启动器内应用 | 可实现应用的移动与换位 |
