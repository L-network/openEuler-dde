# 鼠标右键-发送到桌面

| 编号 | 步骤                                        | 预期                                            |
| ---- | :-------------------------------------------| ----------------------------------------------  |
| 1    | 选择谷歌浏览器，右键，选择发送到桌面        | 桌面新增谷歌浏览器的图标                        |
| 2    | 打开启动器选择计算机，右键                  | 没有发送到桌面选项，替换为了“从桌面移除”选项    |
| 3    | 选择从桌面移除                              | 桌面没有了计算机的图标                          |
