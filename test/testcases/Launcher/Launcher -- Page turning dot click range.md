# 启动器--翻页圆点点击范围

| 编号 | 步骤                                                         | 预期                               |
| ---- | ------------------------------------------------------------ | ---------------------------------- |
|1 | 打开启动器-全屏自由排序 |  切换成功 |
| 2 | 点击下方第一个小圆点范围区域 | 点击无反应 |
| 3 | 点击下方第二个小圆点范围区域  | 切换到第二页 |
| 4 | 中间分界线 | 关闭启动器 |
