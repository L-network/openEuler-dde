# 切换启动器模式

| 编号 | 步骤                                                   | 预期                     |
| ---- | :----------------------------------------------------- | ------------------------ |
| 1    | 打开启动器，显示为全屏窗口时，点击右上角的切换窗口图标 | 启动器被切换成小窗口模式 |
| 2    | 打开启动器，显示为小窗口时，点击右上角的切换窗口图标   | 启动器被切换成全屏模式   |
| 3    | 重复上面1-2步骤分别为3次                               | 每次切换都正常           |
