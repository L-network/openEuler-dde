# 打开剪贴板

前置条件
1.剪贴板已经打开
2.剪切板中存在文本，文件，图片内容

| 编号 | 步骤                                                   | 预期                     |
| ---- | :----------------------------------------------------| ------------------------ |
| 1    | 输入“Ctrl+Alt+V” | 剪贴板自动关闭                          |
| 2    | 鼠标连续单机2次剪贴板中的任意剪切内容        | 剪贴板自动关闭，并选中复制此剪贴块内容   |
| 3    | 鼠标单击非剪贴板区域       | 剪贴板自动关闭   |
