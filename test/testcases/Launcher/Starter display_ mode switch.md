# 启动器显示模式切换

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 点击启动器图标 | 进入启动器界面 |
| 2    | 点击右上角缩放按钮 | 启动器可正常进入全屏模式或恢复默认 |
| 3    | 启动器全屏后点击左上角按钮 | 启动器显示模式切换为按模块显示 |
| 4    | 点击屏幕上方按钮  | 跟根据模块筛选同类型应用 |
