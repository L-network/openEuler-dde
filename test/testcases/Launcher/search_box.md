# 搜索框的光标聚焦

| 编号 | 步骤                        | 预期                                                 |
| ---- | :---------------------------| ---------------------------------------------------- |
| 1    | 打开启动器                  | 可以正常打开                                         |
| 2    | 鼠标点击搜索框              | 光标驻留在搜索框内，只有搜索图标，“搜索”两个字消失   |
| 3    | 退出启动器再次进入          | 搜索框恢复默认状态，有“搜索”两个字显示在搜索框及图标 |
