# 右键菜单-设置壁纸

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 在桌面空白处，右键-设置壁纸 | 进入壁纸设置页面 |
| 2    | 选择一张图片 | 背景图片变为所选图片 |
| 3  | 点击仅设置桌面              | 退出壁纸设置界面，桌面壁纸变为所选图片 |
| 4    | 点击仅设置锁屏     | 退出壁纸设置界面，锁屏壁纸变为所选图片 |
