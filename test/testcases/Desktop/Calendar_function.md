# 日历功能测试
前置条件<br>
#图形化环境<br>

| 编号 |                              |                                                   |
|------|------------------------------|---------------------------------------------------|
|   1  | 任务栏点击日历 | 可正常打开日历视图  |
|   2  | 日历页面 | 显示正常，且显示日期和当前系统时间一致 |
|   3  | 日历左上方有“年”，“月”，“周”，“日”分类 |  点击可以随意切换视图  |
|   4  | 点击搜索框，随意搜索节日，如“端午” |  视图月份不会变化，右侧出现端午节对应的日期 |
|   5  | 点击日历右上角“+”      |  可新建日程  |
