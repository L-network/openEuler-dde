# 任务栏-修改大小

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 鼠标移置任务栏右键,选择任务栏设置| 打开任务栏设置页面 |
| 2    | 选择设置任务栏大小 | 跟随鼠标拖动任务栏可设置大小  |
| 3    | 点击右上角菜单退出 |退出成功  |
| 4    | 重启计算机查看任务栏状态  |状态为重启前设置的大小  |
