# 桌面默认显示

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 使用系统已存在的帐户，图形化界面登入系统，查看桌面默认显示图标 | 桌面默认有且仅有两个图标：计算机、回收站 |
