# 亮度调节百分比显示

| 编号 | 步骤                                   | 预期                         |
| ---- | -------------------------------------- | ---------------------------- |
| 1    | 打开控制中心-显示，查看当前亮度显示    | 显示屏幕名称，及当前调整屏幕亮度百分比数据             |
| 2    | 拖动亮度调节控件，查看亮度调节范围      | 调整屏幕亮度范围：10%-100%              |
| 3    | 拖动亮度调节控件，查看调整间隔| 屏幕发生多次旋转  | 调整间隔最小支持1%    |
