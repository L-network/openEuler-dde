# 添加快捷方式

| 编号 | 步骤                                                         | 预期                                     |
| ---- | ------------------------------------------------------------ | ---------------------------------------- |
| 1    | 控制中心-键盘语言-快捷键，点击添加按钮                       | 进入快捷键添加界面                       |
| 2    | 添加快捷键界面显示                                           | 名称、命令、快捷键输入框、取消和添加按钮 |
| 3    | 输入名称和命令和快捷键，点击添加<br/>如：名称：文档查看器 命令：deepin-reader 快捷键super+R | 添加成功，返回到快捷键首页               |
| 4    | 使用自定义快捷键super+R                                      | 打开文档查看器                           |
