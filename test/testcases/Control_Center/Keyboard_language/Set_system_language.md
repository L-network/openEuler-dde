# 设置系统语言

| 编号 | 步骤                                             | 预期                                                         |
| ---- | ------------------------------------------------ | ------------------------------------------------------------ |
| 1    | 控制中心-键盘语言-系统语言，搜索American English | 显示American English                                         |
| 2    | 点击American English                             | 系统通知提示：正在切换系统语言并安装缺失的语言包，请稍后.... |
| 3    | 语言切换成功                                     | 弹出系统提示：语言切换成功，请注销后重新登录                 |
| 4    | 注销重新登录，检查系统语言切换是否成功           | 语言切换成功，系统变成English                                |
