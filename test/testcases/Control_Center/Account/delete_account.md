# 删除帐户（当前登陆）

| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1    | 启动控制中心-账户” | 显示当前已有账户，其中带绿色的是当前登陆账户                                             |
| 2    | 当前为root或者管理员账户登陆，点击其他账户，选择“删除账户”                | 弹出删除确认框 |
| 3    | 点击删除         | 该用户已被删除 |
