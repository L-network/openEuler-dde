# 全名设置

| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1    | 启动控制中心-账户” | 显示当前已有账户，其中带绿色的是当前登陆账户                                             |
| 2    | 选择用户，然后点击设置全名              | 输入框内可以编辑内容 |
| 3    | 编辑好全名后，点击其他地方      | 弹出验证框需要验证密码 |
| 4    | 验证密码输入后，验证成功      | 用户全名已修改|
