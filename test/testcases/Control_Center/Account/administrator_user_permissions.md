# 标准用户权限

| 编号 | 步骤                                                         | 预期                               |
| ---- | ------------------------------------------------------------ | ---------------------------------- |
| 1    | 打开控制中心-帐户，在帐户列表点击“+”，新建一个管理员Administrator | 新建管理员用户Administrator成功    |
| 2    | 切换用户为管理员用户Administrator                            | 切换Administrator用户成功          |
| 3    | 使用ctrl+alt+t打开终端，输入sudo passwd Administrator        | 管理员用户有sudo权限，命令执行成功 |
