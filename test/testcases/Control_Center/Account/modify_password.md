# 修改密码

| 编号 | 步骤                                                         | 预期                                                |
| ---- | ------------------------------------------------------------ | --------------------------------------------------- |
| 1    | 启动控制中心-账户，在账户列表中选择当前登录帐户，点击修改密码 | 进入密码修改便界面： 当前密码： 新密码： 重复密码： |
| 2    | 输入正确内容，点击确认                                       | 返回到当前账户的详情菜单（三级菜单）                |
| 3    | 锁定系统检查密码修改情况                                     | 使用新密码可以登录成功，旧密码不能使用              |
| 4    | 注销系统测试                                                 | 使用新密码可以登录成功，旧密码不能使用              |
| 5    | 重启系统测试                                                 | 使用新密码可以登录成功，旧密码不能使用              |
