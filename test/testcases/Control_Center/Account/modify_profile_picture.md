# 修改头像

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 启动控制中心-账户，选择当前登录账户，修改头像 | 修改成功，不会弹出密码授权 |
| 2    | 修改其他账户test，点击修改头像                | 弹出密码授权             |
| 3    | 点击取消                                      | 头像修改失败           |
| 4    | 输入正确密码，确认                            | 头像修改成功            |
