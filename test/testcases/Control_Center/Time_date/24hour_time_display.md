# 时间日期--24小时制

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    |  打开控制中心，进入时间和日期模块  |         |
| 2    |  检查当前24小时制的开关状态  |     默认开启，时间为24小时制显示    |
| 3    |  点击开关按钮  | 开关关闭，时间变成12小时制，控制中心-时间日期展示 以及 插件栏时间日期同步展示 |
| 4    |  再次点击开关按钮  | 开关打开，时间变成24小时制，控制中心-时间日期展示 以及 插件栏时间日期同步展示  |
| 5    |  多次重复打开关闭  |  均能正常打开和关闭，时间制显示正确  |
