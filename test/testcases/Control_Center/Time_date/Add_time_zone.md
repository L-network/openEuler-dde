# 添加时区

| 编号 | 步骤                                 | 预期                                                 |
| ---- | ------------------------------------ | ---------------------------------------------------- |
| 1    | 检查页面显示                         | “时区列表”默认被选中，深蓝色背景，且时区列表默认为空 |
| 2    | 点击右侧底部的+按钮                  | 弹出地图添加时区界面                                 |
| 3    | 在地图上点击某一时区，点击取消按钮   | 该地区没有被添加到时区                               |
| 4    | 在地图上点击某一时区后，点击确定按钮 | 无需授权，可直接添加到时区列表                       |
