# 自动补全

| 编号 | 步骤                                                        | 预期                                                 |
| ---- | ----------------------------------------------------------- | ---------------------------------------------------- |
| 1    | 选择系统时区-修改系统时区，进入修改页面，搜索栏输入“shang”  | 搜索结果自动补全“shang”开头的城市，如"shanghai"等    |
| 2    | 回车                                                        | 第一个结果的时区会在地图上标识                       |
| 3    | 搜索栏输入“长”                                              | 搜索不到“长”开头的城市，结果不能自动补全             |
| 4    | 回车                                                        | 第一个结果不会在地图上标示                           |
