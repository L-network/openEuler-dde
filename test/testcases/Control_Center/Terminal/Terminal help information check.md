#  终端帮助信息检查

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 右键dock上终端图标  |            打开终端   |
| 2    |点击右菜单栏 |弹出菜单栏
| 3    |选自帮助   |打开帮助信息 |
| 4    | 检查信息内容|  与系统实际相符,无错字 |
| 5    | 点击右上角X |退出帮助页面 |
