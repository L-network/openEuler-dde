#  终端退出

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 点击退出                                      | 所有终端都被关闭   |
| 2    | 打开终端后，按Alt+F4查看软件显示              | 关闭当前窗口 |
| 3    | 打开终端后，按Ctrl+Shift+W查看软件显示        | 关闭当前窗口 |
| 4    | 打开终端后，按Ctrl+D查看软件显示              | 关闭当前窗口 |
