# 字体

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 使用系统已存在的帐户，图形化界面登入系统，点击控制中心->个性化->字体 | 右侧从上到下依次有<字号>，<标准字体>，<等宽字体>3个可选项 |
| 2    | 查看<字号>功能 --滚动调节字号| 文本显示的字号大小发生相应变化 |
| 3    | 查看<标准字体>功能 --切换字体类型 | 文本显示的字体发生相应变化 |
| 4    | 查看<等宽字体>功能 --切换字体类型 | 所选字体类型变成等宽 |
