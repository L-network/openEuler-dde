# 勿扰模式开启

前置条件
当前时间不在22：00到7：00以内

| 编号 | 步骤                                               | 预期                               |
| ---- | --------------------------------------------------- | ---------------------------------- |
| 1 |打开控制中心－通知－勿扰模式| 勿扰模式选项默认勾选“从22：00到7：00”，菜单栏中的通知中心图标改变样式|
|  2  | 点击截图录屏应用通知选项  |显示四个选项开关默认全部勾选  |
|  3 | 在桌面使用ctrl+alt+A快捷键截图  |  通知消息横幅显示，有声音，通知出现在“通知中心”  |
