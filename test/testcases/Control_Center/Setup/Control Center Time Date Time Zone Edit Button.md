# 标题： 控制中心-时间日期-时区编辑按钮
* 优先级：2

前置条件

控制中心可以正常打开

| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1   | 时区列表为空时|不展示任何信息|
| 2   | 时区列表不为空时点击编辑按钮 |每个时区列表后出现删除按钮，此时”编辑“显示为”完成“|
| 3   |点击完成按钮  |按钮变成编辑|
| 4   |点击时区列表后面的删除按钮|	该时区从列表消失|
