# 标题：控制中心-显示-旋转
* 优先级：1

| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1    | 进入屏幕旋转界面，点击方向下拉框 | 弹出旋转角度为 90/180/270 |
| 2    |进入屏幕旋转界面，点击方向下拉框,分别旋转90/180/270 |屏幕发生多次旋转 |
| 3    |旋转状态被保存，返回"显示"页面 | 旋转状态被保存，返回"显示"页面  |
