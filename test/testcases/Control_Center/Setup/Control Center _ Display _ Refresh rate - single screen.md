# 标题： 控制中心_显示_刷新率-单屏
* 优先级：1

| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1    |进入控制中心-显示，检查刷新率模块 |在显示的二级菜单，以下拉选的方式展示，点击右侧向下的箭头，展示该显示器适用的刷新率（xrandr可查询），默认最优选，仅单选 |
| 2    | 选择其中一个选项   |弹窗提示保存或还原 |
| 3    |  点击保存     | 刷新率修改生效  |
| 4    | 重复步骤二后关闭弹窗/点击还原/等待15s倒计时  | 刷新率被还原  |
