#  终端快捷键检查

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    |右键dock上终端图标  |            打开终端   |
| 2 |点击右菜单栏 | 弹出菜单栏 |
| 3 |选自设置 | 打开基础设置页面 |
| 4 |选择快捷键 | 展示当前快捷键 |
| 5 |遍历所有快捷键 | 快捷键使用成功 |
| 6 |修改某个快捷键设置 | 修改成功、能正常使用 |
