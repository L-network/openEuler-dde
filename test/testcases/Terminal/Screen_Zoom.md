#  屏幕缩放设置

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 点击鼠标右键进入显示设置  |    	显示设置界面成功打开   |
| 2 | 滑动屏幕缩放滑块 | 可从1.0滑动到1.25（或1.25滑到1） |
| 3 | 查看屏幕显示 | 屏幕弹出通知：屏幕缩放需要注销生效 |
| 4 | 点击立即注销 | 系统注销成功 |
| 5 | 重新进入系统后查看系统桌面显示 | 屏幕成功被缩放显示 |
