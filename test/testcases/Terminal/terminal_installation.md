#  终端安装

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 右键dock上终端图标  |    	打开终端   |
| 2    |手动下载软件包成功后，通过终端yum install XXX进行软件安装|可以安装成功
| 3    |安装成功后查看软件是否显示正常，软件图标是否显示正常|软件显示正常，图标显示正常 |
| 4    | 软件安装成功后，再次用终端进行安装，查看软件是否依然能够安装成功|  提示软件已安装|
