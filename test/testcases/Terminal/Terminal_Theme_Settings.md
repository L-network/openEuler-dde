#  终端主题设置

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 右键dock上终端图标  |    	打开终端   |
| 2    |点击右上角菜单栏  |弹出菜单栏
| 3    |依次切换主题   |切换成功 |
| 4    | 点击右上角X |  退出终端 |
| 5    | 再次打开终端 | 保留上次设置的主题  |
