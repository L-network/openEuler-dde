#  雷神窗口显示 

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 右键dock上终端图标，选择雷神终端   |    	打开雷神终端   |
| 2 | 检查默认雷神窗口显示 | 使用屏幕的1/3的高度；标签工作区等按钮显示在下方 |
| 3 | 鼠标拖拽窗口底部 | 可以修改窗口高度 |
| 4 | 检查最大窗口高度 | 最大高度不超过屏幕高度的2/3 |
| 5 | 退出雷神窗口（Alt+F2），再次打开 | 记录上次修改的高度 |
