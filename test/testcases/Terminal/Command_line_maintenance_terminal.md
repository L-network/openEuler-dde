#  命令行维护终端

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 打开任意文件夹，右键选择终端内打开  |  打开终端在当前文件夹路径下 |
| 2 | 点击终端右上角菜单，选择设置 | 跳转至设置界面 |
| 3 | 修改字体 | 终端内字体修改成功 |
| 4 | 修改字体大小 | 终端内字体修改成功 |
| 5 | 关闭终端再次打开 | 保留之前的设置 |
