# 通知中心开关
* 优先级: 1

前置条件

切换到桌面（使用快捷键super+d）

| 编号 | 步骤                           | 预期                 |
| ---- | :------------------------------| -------------------- |
| 1    | dock区点击通知中心图标         | 通知中心被打开 |
| 2    | 点击右上角的“扫把图标”         | 通知中心消息被全部清除 |
| 3    | 点击dock通知中心图标           | 通知中心被打开       |
| 4    | 再次点击dock通知中心图标       | 通知中心关闭|
| 5    | 打开通知中心后点击其他空白处   | 通知中心隐藏 |
