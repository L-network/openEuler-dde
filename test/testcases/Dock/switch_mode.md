# 切换任务栏模式

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------- | -------------------- |
| 1    | 鼠标放置任务栏，点击右键选择模式为“时尚模式” | 任务栏切换为时尚模式 |
| 2    | 鼠标放置任务栏，点击右键选择模式为“高效模式” | 任务栏切换为高效模式 |
| 3    | 重复上面1-2步骤2次                           | 每次切换都正常       |
