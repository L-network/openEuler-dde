#标题：应用的添加与移除
#优先级：1
| 编号 | 步骤                                | 预期                                                         |
| ---- | :---------------------------------- | ------------------------------------------------------------ |
| 1    | 通过启动器发送到任务栏的应用 | 在应用区域依次排序 |
| 2    | 通过从桌面拖拽应用图标到应用区域 | 插入到鼠标当前位置 |
| 3    | 通过右键菜单，选择“移除驻留”将应用移出任务栏 | 应用可以从任务栏移除|
| 4    | 通过将图标拖拽出任务栏达到3倍任务栏高度 | 应用可以从任务栏区域移除|
