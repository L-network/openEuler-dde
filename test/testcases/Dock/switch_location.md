# 切换任务位置

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------- | -------------------- |
| 1    | 鼠标放置任务栏，点击右键选择位置为"上" | 任务栏显示在桌面上方 |
| 2    | 鼠标放置任务栏，点击右键选择位置为"左" | 任务栏显示在桌面左方 |
| 3    | 鼠标放置任务栏，点击右键选择位置为"右" | 任务栏显示在桌面右方 |
| 4    | 鼠标放置任务栏，点击右键选择位置为"下" | 任务栏显示在桌面下方 |
