# 切换任务栏状态

| 编号 | 步骤                                         | 预期                 |
| ---- | -------------------------------------------- | -------------------- |
| 1    | 鼠标放置任务栏，点击右键选择状态为“一直显示” | 任务栏会一直显示     |
| 2    | 鼠标放置任务栏，点击右键选择状态为“一直隐藏” | 任务栏会隐藏，鼠标移 | 
|      |                                              |  动到任务栏位置时才会|
|      |                                              |  显示	             |
| 3    | 重复上面1-2步骤2次                           | 每次操作都正常       |
