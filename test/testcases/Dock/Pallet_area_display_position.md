# 托盘区域显示位置

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 检查时尚模式托盘区域显示位置   |   托盘区域居右显示，应用图标从右往左边撑      |
| 2    | 检查高效模式托盘区域显示位置  |    托盘区域居右显示，应用图标从右往左边撑     |
| 3    | 点击小箭头  |  图标可以展开和收起        |
| 4    | 拖拽图标到箭头右边，再点击小箭头  |   	拖拽到右边的图标不会被收起       |
