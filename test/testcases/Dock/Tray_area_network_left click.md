# 托盘区域—网络——左键单击

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------- | -------------------- |
| 1    | 鼠标左键单击任务栏托盘区域网络图表 | 弹出网络列表 |
| 2    | 检查有线网络列表显示 | 有线网络列表顶部显示网络开关，列表显示所有可连接的有线网络设备，无效有线网络不展示； |
| 3    | 点击列表项 | 则请求连接网络，根据网络状态显示为已连接/连接中的图标|
| 4    | 有线网络关闭或无网络连接状态时点击网络图标 | 唤起控制中心-网络设置|
