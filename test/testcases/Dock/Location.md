# 切换任务栏模式

| 编号 | 步骤                                         | 预期                 |
| ---- | :------------------------------------------- | -------------------- |
| 1    | 鼠标右击任务栏                               |会弹出任务栏设置模块  |
| 2    | 鼠标分别选中上、下、左、右                   |任务栏也会对应在桌面上、
|      |                                              | 下、左、右显示       |
| 3    |重复操作3次                                   |每次任务栏位置都正确  |
